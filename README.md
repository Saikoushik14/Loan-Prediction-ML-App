# 💼 Loan Prediction ML Web App

A Machine Learning-powered web application that predicts loan-related outcomes based on user financial data. Built using **Streamlit**, this app allows users to upload datasets, preprocess data, train models, and evaluate performance interactively.

---

## 🚀 Live Demo

👉 https://loan-prediction-ml-app-2ycfrupd75qesx5tiyexmz.streamlit.app/
---

## 📌 Features

* 📂 Upload custom CSV datasets
* 🧹 Automatic data preprocessing

  * Missing value handling
  * Outlier treatment
* 🔄 Data transformation

  * Encoding categorical features
  * Feature scaling
* 🤖 Model training with multiple algorithms:

  * K-Nearest Neighbors (KNN)
  * Linear Regression
  * Decision Tree Regressor
* 📊 Model evaluation:

  * Mean Squared Error (MSE)
  * R² Score
* 📉 Visualization:

  * Boxplots
  * Elbow method (for KNN tuning)

## ⚙️ Installation & Run Locally

1. Clone the repository:

```
git clone https://github.com/Saikoushik14/Loan-Prediction-ML-App
cd Loan-Prediction-ML-App
```

2. Create virtual environment (recommended):

```
python -m venv venv
venv\Scripts\activate
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```

---

## 📊 How It Works

1. Upload your dataset (CSV format)
2. The app automatically:

   * Cleans missing values
   * Handles outliers
   * Encodes categorical data
   * Scales numerical features
3. Select a machine learning model
4. Train and evaluate performance instantly
